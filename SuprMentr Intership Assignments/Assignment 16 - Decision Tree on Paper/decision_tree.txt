1. INTRODUCTION

A Decision Tree is a supervised machine learning algorithm used for classification and decision-making. It uses a tree-like structure to represent decisions and their possible outcomes.

--------------------------------------------------

2. OBJECTIVE

To design a decision tree that predicts whether a person should play outside based on weather conditions.

--------------------------------------------------

3. FEATURES USED

- Outlook (Sunny, Rainy, Overcast)
- Temperature (Hot, Mild, Cool)
- Humidity (High, Normal)
- Wind (Weak, Strong)

--------------------------------------------------

4. DECISION TREE STRUCTURE

                Outlook
              /    |     \
          Sunny  Overcast  Rainy
           /         |        \
      Humidity      Yes      Wind
       /   \                 /   \
    High  Normal        Strong  Weak
     No      Yes          No     Yes

--------------------------------------------------

5. EXPLANATION

- If Outlook is Overcast → Play = Yes

- If Outlook is Sunny:
  - Humidity High → No
  - Humidity Normal → Yes

- If Outlook is Rainy:
  - Wind Strong → No
  - Wind Weak → Yes

--------------------------------------------------

6. CONCLUSION

Decision Trees help in making decisions based on conditions. They are easy to understand and useful for real-life problems like weather-based predictions.
